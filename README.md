## Version

5113b52715be4c01a8f6db83d38b8d75ee924c88

## Prerequisites

-   Node.js ([Download here](https://nodejs.org/) - LTS version recommended)
-   npm (automatically installed with Node.js)

## Run Instructions

Install dependencies:

```bash
npm install
```

Build the project (optional):

```bash
npm run build
```

Run tests:

```bash
npm run test
```
